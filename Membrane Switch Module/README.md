## Components:
- 1 x Arduino Mega 2560 R3
- 1 x Membrane Switch Module
- 1 x LCD Display
- 1 x 10k Ohm Potentiometer
- M-M Jumper Wires

## Features:
- The input from the Membrane Switch Module is displayed to the LCD Display.
