## Components:
- 1 x Arduino MEGA2560 R3
- 1 x Passive Buzzer
- 1 x 30 Row Breadboard
- 6 x M-M Wires
- 2 x M-F Wires

## Features:
- Tones of the G4 Mixolydian scale can be played by adjusting the potentiometer.

## Learning Log:
- Using the tone() function to play tones.
