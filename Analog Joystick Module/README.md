## Components:
- 1 x Arduino Mega 2560 R3
- 1 x Analog Joystick Module
- 1 x LCD 1602 Display (Non-i2c)
- 1 x Potentiometer
- M-F and M-M Jumper wires

## Features:
- The x and y values of the joystick are displayed on the LCD 1602 display.
