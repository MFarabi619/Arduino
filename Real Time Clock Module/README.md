## Components:
- 1 x Arduino Mega 2560 R3
- 1 x DS1307 Real Time Clock Module
- 1 x LCD 16x2 Display
- 1 x 10k Potentiometer

## Features:
- The time and date outputted by the clock module is displayed on the LCD display.
