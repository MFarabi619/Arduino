## Components:
- Arduino Mega2560 R3
- Ultrasonic Sensor HC-SR04
- 1 x Blue Backlight LCD 1602 (non-i2c).
- 1 x Breadboard
- 18 x M-M Wires

## Features:
- The distance detected by the ultrasonic sensor is shown on the LCD 1602 display.
